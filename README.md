# AR Museum Application

## Overview

This project is an Augmented Reality (AR) application designed for small museums to enhance visitor experiences. The application overlays museum artefacts with interactive digital content such as additional information, animations, and audio, making exhibits more engaging and educational.

## Features

* Augmented Reality overlays for museum artefacts
* Interactive content including text, animations, and audio
* Local tracking of visitor engagement
* Records time spent interacting with exhibits
* Logs the number of visits per user
* Privacy-focused, local-first design

## Offline Support

The application is designed to function without an internet connection, making it suitable for small museums with limited or no connectivity.

## Project Status

This project is currently under development and is a work in progress.

## Purpose

The goal of this project is to provide a simple, affordable, and privacy-conscious AR solution that helps small museums create immersive visitor experiences without requiring complex infrastructure.
